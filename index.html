Introduction

LoLUpdater is a tool for the game League of Legends that increases the performance of the game, launcher and client.

Installation

.Net 4.5.2

Download and Install .Net 4.0 (Windows XP ONLY)

Launch from your LoL directory, Examples:

C:\Riot Games\League of Legends

Garena 64bit computers

C:\Program Files (x86)\GarenaLoL\GameData\Apps\LoL

C:\Program Files (x86)\GarenaLoLPH\GameData\Apps\LoLPH

C:\Program Files\GarenaLoL\GameData\Apps\LoL
