Introduction

LoLUpdater is a tool for the game League of Legends that increases the performance of the game, launcher and client.

Download

LoLUpdater LoLUpdater for XP

Installation

.Net 4.5.2 .Net 4.0 for XP

Launch from your LoL directory, Examples:

Standard LoL directory C:\Riot Games\League of Legends
Garena (32-Bit) C:\Program Files (x86)\GarenaLoL\GameData\Apps\LoL
C:\Program Files (x86)\GarenaLoLPH\GameData\Apps\LoLPH
Garena (64-Bit) C:\Program Files\GarenaLoL\GameData\Apps\LoL
C:\Program Files\GarenaLoLPH\GameData\Apps\LoLPH
